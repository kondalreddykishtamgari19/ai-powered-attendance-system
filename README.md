# AI-Powered Attendance Management System

## Overview

A full-stack AI-based attendance management system using facial recognition technology.

This project uses:
- MERN Stack
- OpenCV
- Face Recognition
- MongoDB
- React Dashboard

The system automatically detects and recognizes student faces and marks attendance in real-time.

---

## Features

### Authentication
- User Registration
- User Login
- Logout System
- Protected Dashboard

### AI Features
- Face Detection
- Face Recognition
- Duplicate Face Prevention
- Automatic Attendance Marking

### Dashboard Features
- Total Students
- Present/Absent Statistics
- Weekly Attendance Graph
- Monthly Attendance Graph
- Student Management
- Attendance Records

## Installation

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
node server.js
```

### AI System

```bash
cd ai

venv\\Scripts\\activate

python attendance_system.py
```
